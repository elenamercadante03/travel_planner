About Travel Planner

Travel Planner is a web application developed with Flask and SQLAlchemy.

Users can create trips, manage destinations and retrieve real-time information from external APIs.

Technologies Used

  • Python
  • Flask
  • SQLAlchemy
  • SQLite
  • HTML/CSS

External APIs

  • OpenStreetMap (Geolocation)
  • Open-Meteo (Weather)
  • REST Countries (Country Information)
