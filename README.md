Travel Planner is a web application developed with Flask and SQLAlchemy.

The application allows users to register, log in and manage their personal travel plans.

Users can create trips, add destinations and retrieve real-time information through external REST APIs.

The project integrates three external APIs:

1. OpenStreetMap (Nominatim) for geographical coordinates.
2. Open-Meteo for weather information.
3. REST Countries for country information.

Data is stored in a SQLite relational database and managed through SQLAlchemy ORM.

The application follows a multi-page Flask architecture using HTML templates and custom CSS styling.
